Dex de colección — Configuración del bot

El bot invoca un elemento al azar cuando alguien escribe el comando de spawn, y lo guarda en la colección de quien lo reciba.
Los usuarios escriben este comando para invocar un elemento al azar.
Los usuarios escriben este comando para ver lo que han coleccionado.
Variables disponibles: {icon} {numero} {nombre} {rareza} {descripcion} {frase} {imagen}. Deja vacía una línea para omitirla.
Elementos del Dex (JSON)
Pega un array JSON con tus elementos. Cada uno admite: name, icon (un emoji, opcional), rarity (common/uncommon/rare/epic/legendary), description, quote (opcional), imageUrl (opcional), weight (1-100, qué tan seguido aparece).
